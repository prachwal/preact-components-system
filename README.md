# Preact Components System

A system of reusable Preact components.

## Installation

```bash
npm install preact-components-system
```

## Usage

```tsx
import { App } from 'preact-components-system';

function MyApp() {
  return <App />;
}
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
